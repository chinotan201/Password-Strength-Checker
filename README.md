# â­ **DESCRIPTION**

**Password Strength Checker 9000** is a Python-based security awareness tool designed to help users evaluate the strength and resilience of their passwords in an entertaining and visually engaging format.

The program analyzes password complexity using **regex-based detection**, provides real-time feedback, and estimates crack difficulty through a humorous, hacker-style interface powered by `pyfiglet` and `termcolor`.

This tool is perfect for:
- **Cybersecurity beginners**
- **Students learning Python**
- **Anyone who wants to improve password habits**
