PASSWORD CHECKER 9000 is a Python-based security awareness tool built to help users evaluate the strength and resilience of their passwords in a fun, interactive, and visually engaging way.

The program analyzes password complexity using regex-based pattern detection, provides real-time feedback, and estimates crack difficulty in an entertaining, hacker-themed interface.

Designed for education, security training, and personal self-assessment, this tool helps individuals understand common weaknesses in password creation and highlights best practices for strong, secure authentication.

Key Features:

Password Complexity Analysis
Evaluates length, character variety, special symbols, and known weak patterns.

Crack-Time Estimation
Provides humorous yet educational strength ratings based on basic heuristic models.

Color-Coded Security Feedback
Uses clear visual indicators to show which password requirements are met.

ASCII-Art Interface
Delivers a stylish, hacker-inspired terminal experience powered by pyfiglet and termcolor.

Beginner-Friendly Codebase
Ideal for students, cybersecurity beginners, and Python learners.
